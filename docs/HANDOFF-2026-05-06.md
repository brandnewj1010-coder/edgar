## 6. 트러블슈팅

### `.git/HEAD` 누락 / "not a git repository" 오류
Google Drive sync가 .git 작은 파일을 잠그며 발생. PowerShell:

```powershell
[System.IO.File]::WriteAllText("$PWD\.git\HEAD", "ref: refs/heads/feature/phase-0-renewal-base`n", (New-Object System.Text.UTF8Encoding $false))
git status
```

### `npm install` 후 `node_modules/.bin/` 비어 있음
부분 설치. Drive sync 일시정지하고 깨끗하게:

```bash
rm -rf node_modules package-lock.json
npm install
ls node_modules/.bin/vite*    # vite.cmd 보이는지 확인
```

### 빌드에서 Sankey `label` prop 타입 에러
`src/components/SankeyViz.tsx`의 `label={(node) => ...}` 라인에 `label={((node) => ...) as any}` 한 번만 캐스팅. 또는 props 분해 시 `node`에 `as { label?: string; value?: number }` 부착.

### feature 브랜치가 사라졌거나 main에 있음
```bash
git branch -a              # 로컬·원격 브랜치 다 보기
git checkout feature/phase-0-renewal-base
```
원격에도 없고 로컬에도 없으면 `.git/refs/heads/feature/phase-0-renewal-base` 파일이 있는지 확인. 없다면 main에서 다시 분기:
```bash
git checkout -b feature/phase-0-renewal-base
```
이 경우 코드 변경(uncommitted)이 살아 있으면 그대로 남고, 사라졌으면 §2 의 변경 목록을 보고 plan(`docs/plans/2026-05-06-pr1-phase-0-renewal-base.md`)을 다시 실행해야 함.
